# Remote Authentication Dial-In User Service (RADIUS) 


RADIUS is a client/server protocol that runs in the `application layer`, and can use either TCP or UDP.

RADIUS is an AAA (authentication, authorization, and accounting) protocol that manages network access, it is a security protocol used in the AAA framework to provide centralized authentication for users who want to gain access to the network. 


It uses UDP as a transmission protocol. 

It uses UDP port number `1812` for authentication and authorization and 1813 for accounting
