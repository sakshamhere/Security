# WinRM ( Windows Remote Management )

Windows remote management can be used to facilitate remote access over `HTTP(S)`

Typically uses `TCP port 5985 and 5986 (HTTPS)`

`WinRM` was made to make system adminstrators life easy

`WinRM` is typically used for 

- remotely access and interact windows hosts on local network
- remotrly access and execute commands on windows system
- remotely manage and cinfigure windows system

Authentication for WinRm can be done using username and password & username and password hash

basically same as RDP
