
# FTP Bounce Attack

An FTP Bounce attack is an old type of network attack that is performed on FTP servers to send outbound traffic to typically another server in the network.

The attacker issues a `PORT` command and tricks the FTP connection to execute commands and extract sensitive information from another device instead of the intended server. 

This enables a way for the attacker to communicate to a third device on the network and gain unauthorized access to sensitive information from that device.
