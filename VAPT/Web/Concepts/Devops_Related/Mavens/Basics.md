
# What is Mavens?
https://www.youtube.com/watch?v=gaN1DUJmkz8

Maven is Powerful Project Management tool based on POM (Project Object Model)

There exist a file POM.xml, where dev can specify the dependency required for project, then mavens will download the JAR file for that dependency 

So the dev is not required to go on internet and search files or update of anything, mavens can take care everything

Maven is used to create project build ie jar/war file etc, dependency and documentation, It simplifies build process like ANT which was used before, Mavens is more advanced than ANT, another alternative to Mavens is Gradle

so people mostly use Mavens or Gradle

conclusion - mavens can provide us Dependencies, Build, Documentation, project structurem, Releases etc on its own 
             which helps us to focus on logic
