# Dynamic Host Configuration Protocol (DHCP server)
https://efficientip.com/glossary/what-is-dhcp-and-why-is-it-important/

Dynamic Host Configuration Protocol (DHCP) is a network protocol used to automate the process of configuring devices on IP networks, thus allowing them to use network services such as DNS, NTP, and any communication protocol based on UDP or TCP.

A DHCP server dynamically assigns an IP address and other network configuration parameters to each device on a network so they can communicate with other IP networks. 


# DHCP on Home Networks
https://stevessmarthomeguide.com/understanding-dhcp-home-networks/

DHCP allows devices to join a home network without the need for any device setup.

The DHCP protocol consists of two components.

    A DHCP client
    A DHCP server


All modern operating systems come equipped with a DHCP client, and by default, they are all configured to use DHCP.

On home networks the `DHCP server` is on  the `home router` or home hub.


Note - If you are thinking of using another device to be a DHCP server it is important to understand that you can only have one DHCP server on a network.
